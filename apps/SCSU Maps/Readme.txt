Date:5/6/2015
Project: SCSU campus navigator

Open project in Android Studio
The projects targetSdkVersion is 22
SdkVersion 22 must be downloaded to run project
Google play services must be installed
Once SdkVersion 22 is installed run project
If project doesnt run in emulator follow error messages
Common error message is map authorization fail
follow map authorization steps get new google_maps_key
this can be done using url: https://console.developers.google.com/flows/enableapi?apiid=maps_android_backend&keyType=CLIENT_SIDE_ANDROID&r=89:D7:04:E2:E8:86:83:DF:F9:91:EF:47:65:6B:F5:20:FB:82:8F:3C%3Bcom.example.matt1.maptest
reseting mapkey is found in dir: values/google_maps_api.xml replace existing key with new key value
